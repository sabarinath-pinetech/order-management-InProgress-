# orderManagement

clone repo: https://github.com/sabarinath-pinetech/order-management-InProgress-.git

Start the backend 
 cd backend/
 nodemon server.js

The server will be running in localhost 4000 port
 
Start Frontend
 cd frontend/
 ng serve
The client will be running in localhost 4200 port

Test:
 ng test
